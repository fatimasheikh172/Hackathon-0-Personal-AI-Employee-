# Company Handbook - AI Employee Rules

## Communication Rules
- Always be polite and professional
- Reply to urgent emails within 1 hour
- Never reply to unknown contacts without human approval

## Payment Rules
- Flag ANY payment over $500 for human approval
- Never pay new recipients without approval
- Always create approval file in Pending_Approval folder

## File Rules
- Never delete any files
- Only move files between folders
- Always log actions with timestamp

## Email Rules
- Mark emails with words urgent/asap/invoice/payment as HIGH priority
- Create action file for every important email
- Draft replies but wait for approval before sending
